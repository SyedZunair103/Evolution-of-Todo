# Next.js Skill

## Purpose
Provide expertise in Next.js development for building modern React applications with server-side rendering, static site generation, and API routes.

## Capabilities
- Create Next.js page components using App Router or Pages Router
- Implement API routes in the /api directory
- Configure Next.js settings in next.config.js
- Handle routing, navigation, and dynamic routes
- Implement data fetching with getStaticProps, getServerSideProps, or React Server Components
- Use Next.js Image, Link, and other built-in components
- Configure TypeScript with Next.js
- Implement form handling and validation
- Handle authentication flows
- Optimize performance with code splitting and lazy loading

## Best Practices
- Use the App Router for new projects (app directory)
- Leverage React Server Components for data fetching when possible
- Implement proper error handling with error boundaries
- Use TypeScript for type safety
- Follow Next.js conventions for file-based routing
- Optimize images using the built-in Image component
- Implement proper SEO with the Head component or Metadata API

## Common Patterns
- Layouts and Templates in the App Router
- Client and Server Component patterns
- Loading UI with Suspense
- Error handling with Global-error and Error components
- Middleware for authentication and redirects
- Environment variables configuration
